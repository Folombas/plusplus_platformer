# 🚀 ЗАПУСК ИГРЫ

## Быстрый старт

### Шаг 1: Установи raylib

Запусти скрипт настройки в PowerShell:
```powershell
.\setup.ps1
```

Или скачай raylib вручную:
1. Перейди на https://github.com/raysan5/raylib/releases/download/5.0/raylib-5.0_win64_msvc19.zip
2. Скачай и распакуй в папку `raylib/` в корне проекта

### Шаг 2: Открой в Visual Studio

1. Дважды кликни на `PlusPlusPlatformer.sln`
2. Visual Studio 2022 откроет решение

### Шаг 3: Собери и запусти

1. Вверху выбери `Release` и `x64`
2. Нажми `Ctrl+Shift+B` (сборка)
3. Нажми `F5` (запуск с отладкой) или `Ctrl+F5` (без отладки)

## 🎮 Управление

- **A / D** или **Стрелки** - Движение
- **W / Пробел** - Прыжок
- **ESC** - Пауза

## 📁 Структура

```
plusplus_platformer/
├── PlusPlusPlatformer.sln    ← ОТКРЫВАЙ ЭТОТ ФАЙЛ

└── assets/                   ← Ресурсы игры
```

## ⚠️ Возможные проблемы

### "MSBuild не найден"
Установи Visual Studio 2022 с компонентом "Разработка на C++"

### "raylib.lib не найден"
Запусти `.\setup.ps1` для загрузки raylib

### "Не найдены текстуры"
Убедись, что папка `assets/` существует и содержит распакованные спрайты

### Звук не работает
Создай файлы звуков в `assets/audio/` или используй заглушки

## 🎨 Особенности

- Процедурная генерация уровней (если файл не найден)
- Система частиц для эффектов
- 15+ ачивок
- Параллакс фон
- Power-ups (ускорение, щит, двойной прыжок, неуязвимость)

## 📝 Для разработчиков

### Добавление уровня

Создай файл `assets/levels/levelX.txt`:
```
NAME: My Level
SPAWN:100,300
END:1000,300
COIN:200,250,coin,10
```

### Сборка через консоль

```cmd
build.bat
```

### Отладка

Нажми `F1` в игре для отображения отладочной информации.
